# Healthcare Multi-Clinic Analytics Platform

## Overview

An end-to-end AWS Data Engineering project that processes healthcare appointment data from multiple clinic databases. The project demonstrates batch and real-time data pipelines using AWS services and PySpark.

## Business Problem

Healthcare organizations operate multiple clinics, each with its own database. Running analytics directly on production databases can impact application performance. This project builds a scalable analytics platform that extracts data, processes it, and provides business insights.

## Objectives

- Extract data from multiple clinic databases
- Build an AWS Data Lake
- Transform data using PySpark
- Automate ETL pipelines
- Support real-time analytics
- Build dashboards for business users
